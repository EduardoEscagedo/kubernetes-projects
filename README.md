# Kubernetes Projects

Repositorio con proyectos prácticos de Kubernetes orientados a entornos reales de trabajo.
El objetivo es demostrar conocimientos de operación, despliegue y gestión de clusters Kubernetes
con un enfoque DevOps / Platform Engineer.

##  Enfoque del repositorio
- Kubernetes práctico (no ejemplos académicos)
- Pensado para entornos productivos
- Buenas prácticas de operación y despliegue
- Automatización, observabilidad y GitOps

##  Contenido

###  Cluster & Local Environments
- K8s multi-node
- Configuración base de clusters
- Testing local de componentes de producción

###  Networking
- Ingress Controllers (NGINX)
- Service exposure
- DNS y routing interno

###  GitOps & CI/CD
- GitOps con ArgoCD
- Estructura de repositorios
- Despliegue declarativo de aplicaciones

###  Monitoring & Observability
- Prometheus
- Grafana
- Métricas y dashboards básicos

###  Security & Access
- RBAC
- Service Accounts
- Buenas prácticas de seguridad

##  Tecnologías utilizadas
- Kubernetes
- kubectl
- Helm
- ArgoCD
- Prometheus & Grafana
- Docker

##  Objetivo profesional
Este repositorio forma parte de mi portfolio técnico como:
- DevOps Engineer
- Platform Engineer
- Site Reliability Engineer (SRE)

Los proyectos están diseñados para reflejar escenarios reales que se encuentran en entornos empresariales.
